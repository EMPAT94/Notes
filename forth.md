### Print Hello, World

### WORDs

All definitions in forth are just "WORD"s. To define a specific word to mean something, we use the following form:

`: SPECIFIC SOMETHING ;`

where,  
 `:` = Start definition  
 `SPECIFIC` = Definition name  
 `SOMETHING` = Definition value  
 `;` = End definition
