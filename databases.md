# Databases

## RDBMS Notes

### Normalization

- 1st Form

  - Primary key
  - No repetition of column data
  - No multivalue attrs

- 2st Form

  - No repetition of row data
  - All keys in a table depend on primary key
  - Each table is one logical unit

- 3st Form

  - No dependencies in between columns of same table
  - No derived values
